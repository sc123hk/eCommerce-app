## Introduction
An eCommerce application.

## Tech Stack
- Laravel

## Environment
```
php artisan --version
Laravel Framework 10.3.3
php -v
PHP 8.1.2
```

## Configuration
| file | field | value |
| --- | --- | --- |
| .env | DB_DATABASE | database name |
| | DB_USERNAME | mysql user name |
| | DB_PASSWORD | mysql user password | 

## Setup
```
php artisan migrate
php artisan serve
```
